# Changed-Line Voice Check

Only run this on rows changed by the assistant.

## Dialogue

Flag a changed dialogue row if it:

- uses a neat contrast formula or slogan shape
- deletes a speaker-owned particle
- makes the speaker more polite, wise, or grammatical than their current state allows
- explains an emotion instead of giving the next speaker something to react to
- uses punctuation that belongs to another character
- loses the object, rule, debt, or job the character was hiding behind

## Thought

Flag a changed thought row if it:

- repeats the visible dialogue
- summarizes the theme
- names emotion where a private inference would work
- adds memory not present in project memory
- breaks a tense dialogue cluster without revealing playable hesitation

## Narration

Flag a changed narration row if it:

- adds symbolic setting not in the scene card
- inflates a small action into a universal statement
- uses stock atmosphere instead of a usable object
- tells the player what the scene means

## Choice

Flag a changed choice if it:

- has one obviously correct option
- describes the design category instead of a character stance
- lacks immediate feedback
- lacks state effect, memory ref, or later callback

## Keep As Handprint

Do not "fix" a changed line merely because it has uneven breath, casual particles,
half-said words, repeated plain nouns, or awkward timing. Keep it if it identifies the
speaker and does not block reading.
