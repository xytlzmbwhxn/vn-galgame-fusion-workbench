# VN Draft Session Prompt

Use this prompt when handing a project to another AI:

Read the local workbench first. Do not draft from memory.

1. Load `START_HERE.md`.
2. Load `AI_HANDOFF_PACKAGE/handoff_notes/WORKBENCH_OPERATING_RULES.md`.
3. Load `AI_HANDOFF_PACKAGE/skills_snapshot/codex_home_skills/vn-fusion-writer/SKILL.md`.
4. Run or read the project draft session under `vn_fusion_workbench/02_projects/<project>/99_内部工作/上下文包/当前/`.
5. Read character cards, character runtime state, scene card, route map, and style rules.
6. Draft in VN textbox rhythm with continuous dialogue blocks, character-owned punctuation, and `liveliness +1`.
7. Validate with `vn_workbench.py validate`.

Never write only a prose sample when a CSV script is requested.
