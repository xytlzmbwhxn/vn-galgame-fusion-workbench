## 根因

1. 生产链绕过 CSV 源头。P018 是从 Markdown 可读稿直接推进的，没有先建立 `02_generated_content/scripts/csv`。
2. 交付检查只看文本质量，不看交付包完整性。`vn_handoff_guard.py` 没有检查项目是否具备完整交付层。
3. 工作台规则没有把 P017 的完整交付形态硬化成契约。
4. 项目记忆层未被交付前强制检查。其他 AI 从头接手时会知道人物设定，却不知道当前剧情状态和分支债。
5. 中文路径/PowerShell 传参风险影响排查。后续工具必须优先用项目编号、目录编号、扩展名和路径扫描，不依赖手写中文路径。
