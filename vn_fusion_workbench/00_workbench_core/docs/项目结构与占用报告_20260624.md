## 新增强制工具

| 工具 | 用法 |
|---|---|
| vn_context_bootstrap.py | 写作前运行，生成 runtime/last_context_bootstrap.json，证明 skills/方法卡/风格资产/角色卡已加载。 |
| vn_handoff_guard.py | 交付前运行，检查启动收据、泄露标签、禁用句式、心理口吻、POV、对白 ping-pong 等。 |

## 当前学习结论落点

| 学习内容 | 已落地位置 | 用法 |
|---|---|---|
| 教程：对话/心理/客观事实要区块化，心理通常放在对白块之后 | methods/pov_interiority_control.md、spoken_private_thought_texture.md | 写作前判断心理属于谁、何时插入。 |
| 教程：插入心理要有把握该角色此刻会这样想 | vn_handoff_guard.py | 无 POV 声明或心理太像作者评语会失败。 |
| 私有语料对白块：连续对白和同说话人 run 是常态 | vn_handoff_guard.py、WORKBENCH_OPERATING_RULES.md | 检查纯 ping-pong，要求对白所有权。 |
| 用户反馈：禁令会压扁文笔 | chinese_vn_generation_hard_gates.md、VN文风手印融合资产 | 禁令作为清理层，不作为生成起点。 |
