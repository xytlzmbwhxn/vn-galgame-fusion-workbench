# Changed-Line Voice Integrity Check

## Call This When

Load after revising existing script rows.

## Core Rule

Only changed lines need this pass, but every changed line must keep the character's
voice, relationship state, and scene pressure intact.

## Check Each Changed Row

- Does the row still match the speaker's sentence engine?
- Did punctuation become generic or over-polished?
- Did a lively imperfection get flattened?
- Did the line lose its hidden want, dodge, joke, or pressure?
- Did the revision introduce a forbidden AI-style contrast formula?
- Did the row still preserve current relationship state and memory?
- Did the click function improve?

## Required Revision Note

For material changes, record:

- row id
- old issue
- new click function
- voice reason
- whether character memory needs an update

## Failure Signs

- the revised line sounds more correct but less like the character
- all characters become equally polite or articulate
- the line explains emotion instead of making someone act
- branch or memory references no longer match the row
