# Four-Layer Narrative Integration

## Call This When

Load when checking whether gameplay, plot, character, and theme are connected.

## Four Layers

1. Gameplay action: what the player does.
2. Plot event: what happens in the world.
3. Character pressure: what someone wants, hides, risks, or loses.
4. Theme pressure: what repeated value conflict is being tested.

## Scene Requirement

Every important scene must state:

- player action
- information gained or lost
- relationship cost
- state or route effect
- theme behavior tested
- later callback

## Failure Signs

- gameplay is decorative
- choices do not express theme
- theme is explained after the fact
- character conflict does not alter state
- plot moves but the player did not participate meaningfully
