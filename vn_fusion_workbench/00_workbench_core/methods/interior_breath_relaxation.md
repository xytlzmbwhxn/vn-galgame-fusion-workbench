# Interior Breath Relaxation

## Call This When

Load before drafting or revising Chinese VN/Gal inner monologue when the user says
the psychology feels stiff, dead, too formal, too judgmental, or lacks casual
breath.

Use together with:

- `spoken_private_thought_texture`
- `private_thought_pocket_texture`
- `line_energy_punctuation`
- `character_liveliness_bias`

## Core Rule

Inner thought is not a verdict. It is a small live action inside the viewpoint
character: tense up, buffer, self-correct, joke, dodge, relax, then choose the
next move.

A good casual thought row often has one of these breath shapes:

1. **tighten -> soften**
   - `嗯……当然？它应该没有抗议功能。`
   - `糟糕。嗯，先当作没听见。`
2. **guess -> retreat**
   - `她是在等我问？……算了，问了更像审计。`
3. **rule -> human leak**
   - `流程上应该登记。大概。手写也行。`
4. **object joke -> private cost**
   - `这张便签很脆弱。主要是我也没比它强多少。`
5. **serious word -> small particle**
   - `危险物品。嗯，听起来像我在嫉妒。`

## Relaxation Tools

Use these only when they belong to the current POV character:

- soft starters: `嗯`、`啊`、`诶`、`呃`、`唔`
- looseners: `……`、`好吧`、`算了`、`先这样`、`大概`、`应该`
- particles: `吧`、`嘛`、`啦`、`啊`
- tiny retreat: `当我没说`、`先别看`、`这句撤回`
- practical mask: `登记`、`签收`、`流程`、`预算`、`回收`、`送检`
- half joke: make the object carry the embarrassment for half a row

Do not sprinkle these mechanically. The breath must explain how the character is
recovering from pressure, not decorate a finished sentence.

## Placement

Relaxation is strongest at these timing points:

- after an object seems to "answer" the POV
- after the POV catches themself caring too much
- after a direct question would expose affection, jealousy, or embarrassment
- after the other character's joke lands closer than expected
- immediately before a choice or a procedural reply

## Rewrite Test

For each changed thought row, ask:

- What pressure made the POV tighten?
- Which word, particle, or object lets them loosen?
- Does the relaxed wording still belong to this character?
- Does the next click change because of the thought?

Reject the row if it only adds `？`, `……`, or `嗯` without changing the private
stance.

## Failure Signs

- Every thought is a clean declarative sentence.
- Every repair becomes a rhetorical question.
- Fillers are pasted onto lines that already sound finished.
- The thought explains a mechanism instead of letting the POV recover.
- The casual particle belongs to no specific character.
- The row has relaxation texture but no trigger, cost, or next-click effect.
