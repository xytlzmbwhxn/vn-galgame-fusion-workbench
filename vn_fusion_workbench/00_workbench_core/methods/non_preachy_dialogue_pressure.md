# Non-Preachy Dialogue Pressure

## Call This When

Load before drafting or revising dialogue that risks sounding like advice,
theme explanation, craft theory, or AI-polished talk.

## Core Rule

Dialogue should create an actable problem for the next speaker.

Good dialogue usually makes someone:

- answer or dodge
- accept help or reject debt
- reveal knowledge or pretend not to know
- keep face or lower the mask
- choose public rule or private feeling
- protect someone or expose the truth
- bargain, tease, threaten, stall, confess, or misread

## Rewrite Moves

If a line states an idea, convert it into:

- request
- refusal
- joke
- wrong answer
- bargain
- correction
- object operation
- relation cost
- practical problem
- choice pressure

## Blocked Patterns

- "不是 X，而是 Y"
- "真正的 X 是 Y"
- "重要的不是 X，而是 Y"
- "在某种意义上"
- "我终于明白"
- authorial summary of theme

## Failure Signs

- the line could be spoken by the author instead of the character
- the next speaker has no problem to solve
- the line explains what the player should feel
- characters sound wise, calm, and interchangeable
