# Professional Ritual As Gameplay

## Call This When

The protagonist has a job, social role, duty, service action, or investigation procedure.

## Core Use

Turn theme into repeated playable behavior. The player should not choose abstract values; the player should perform a job action under pressure.

## Procedure

1. Name the repeated work action.
2. Name the object it requires.
3. Name who benefits from doing it properly.
4. Name who is harmed by doing it properly.
5. Make the choice happen through the action, not through a moral label.

## Output Fields

- `professional_ritual`
- required object
- who signs, holds, sees, records, or refuses the object
- variable affected by the action

## Failure Signs

- the job is only background flavor
- choices are "be kind / be strict"
- objects do not move between hands
- no later line remembers the work action
