# Subtext From Want / Need Mismatch

## Call This When

Any scene contains serious dialogue, bonding, accusation, confession, romance, or interrogation.

## Inputs

For each speaker:

- surface want
- hidden need
- unsafe word
- object used as cover
- relationship leverage

## Procedure

1. Let the character talk about the safe object.
2. Keep the unsafe word out of the line.
3. Let the other character answer the hidden pressure, not only the literal sentence.
4. When a character nearly says the unsafe word, interrupt with an object, sound, or work action.

## Output Fields

- `surface_topic`
- `hidden_want`
- `unsafe_word`
- `line_function`

## Failure Signs

- characters explain themselves cleanly
- dialogue only conveys lore
- everyone answers exactly what was asked
- an apology or confession arrives before enough cost
