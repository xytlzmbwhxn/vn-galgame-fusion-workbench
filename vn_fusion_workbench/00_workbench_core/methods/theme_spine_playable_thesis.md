# Theme Spine And Playable Thesis

## Call This When

Load before planning a project, route, chapter, ending, or key scene.

## Core Rule

Theme must become player behavior under cost. Do not explain it as a paragraph.

## Define

- thesis question
- repeated player behavior
- emotional cost
- social cost
- concrete object or ritual
- route arguments
- false solution
- ending payoff

## Scene Test

Ask:

- What does the player choose or avoid?
- What does the choice cost now?
- What debt remains later?
- Which character argues the opposite side?
- What object, name, or routine carries the theme?

## Failure Signs

- the protagonist states the theme cleanly
- choices do not test the theme
- endings only summarize a lesson
- routes differ by romance target but not by argument
