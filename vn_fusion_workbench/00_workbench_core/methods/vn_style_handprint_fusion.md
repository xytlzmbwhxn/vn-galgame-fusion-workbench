# VN Style Handprint Fusion

This method turns learned VN/script corpora into a reusable writing handprint.
It is not a checklist, a quote bank, or a permission to imitate source text.

Use it before serious Chinese Galgame/AVG drafting whenever the workbench has
learned tutorials, full scripts, style samples, or user-provided references.

## Required Style Asset

Load the current handprint asset:

## Purpose

Hard gates prevent obvious failure. The handprint layer improves the prose itself.

Before drafting, extract or select:

- sentence moves: how a line turns into another line
- emotional turns: how pressure changes without being explained
- voice handprints: each character's particles, dodge, clause length, and stress marks
- click rhythm: when to merge, split, hold, or hand off to production
- thought pockets: where private inference belongs and where it becomes overthinking
- branch feeling: how an option leaves debt after it rejoins

For thought pockets, also load `private_thought_pocket_texture`. Style absorption
fails if inner rows only diagnose a character, describe visible action, or make
a balanced author judgment in parentheses.

## Generation Contract

The draft must show these in actual rows, not only in notes:

- dialogue blocks that carry pressure before narration interrupts
- lively but character-specific punctuation, not uniform `？？` or `......`
- thoughts that reveal unsaid cost, not camera-visible description
- narration that gives the screen a usable object, action, or timing cue
- emotional shifts that happen through task, relation, mistake, or choice
- theme that is played through conflict and route debt, not summarized

## Future Intake Rule

When the user adds new tutorials or script text, do not only make summaries or
metrics. Update the handprint layer.

Minimum intake output:

```text
New reusable sentence moves:
New character voice patterns:
New emotional transition patterns:
New punctuation / textbox rhythm lessons:
Conflicts with existing rules:
Merged handprint changes:
What should change in the next draft:
```

Move only reusable craft knowledge into workbench methods or style assets.
Project-specific characters, plot facts, and route state stay inside that
project's `00_project_memory/`.

## Anti-Patterns

Reject the draft or intake if:

- the learned text appears only as a citation or metric
- rows obey length rules but have no emotional turn
- all characters share the same speaking temperature
- punctuation is mechanically added after critique
- thoughts explain what the dialogue already said
- source lines, source scenes, or source character portraits are copied
- the next AI cannot tell which style asset to load
