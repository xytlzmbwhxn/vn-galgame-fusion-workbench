# Theme And Motifs

- Ticket stub: what remains after a route is officially closed.
- Stamp: public recognition, also bureaucratic danger.
- Rain on thermal paper: memory can blur if no one handles it carefully.
- Bell: a small public sound that says a passenger is still here.
- Old route manual: memory stored as procedure.
- Orange umbrella charm:乔萌把沉重的事系成小玩意儿，越可爱越说明她在撑。
