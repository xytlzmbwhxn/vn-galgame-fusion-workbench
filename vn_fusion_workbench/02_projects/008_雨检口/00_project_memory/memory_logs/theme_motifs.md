# Theme And Motifs

## Core Themes

| Theme | Core Question | Characters Touched | Status |
| --- | --- | --- | --- |
| Record vs. person | What does a record protect, and whom does it expose? | 邵宁, 岑遥 | Introduced |
| Name as danger | When is a name proof, and when is it a handle for control? | 岑遥 | Introduced |
| Procedural mercy | Can a rule-bound person help without turning help into another form? | 邵宁 | Developing |

## Motifs

### 湿票根

- Meaning: paper evidence that decays while people hesitate.
- First Appearance: S001, in 岑遥's hand.
- Active Scenes: S001, S002.
- Progression: S001 shows the date; S002 reduces it to a partial number.
- Payoff Target: a later route must recover the missing date from another source.

### 备用伞

- Meaning: station property turned into private debt.
- First Appearance: S002.
- Active Scenes: S002.
- Progression: if 岑遥 takes it, the system has a record of an object but not the reason it was given.
- Payoff Target: later return or failure to return reveals trust.

### 黄线

- Meaning: visible boundary between being recorded and being safe.
- First Appearance: S002.
- Active Scenes: S002.
- Progression: characters learn to use official boundaries against official sight.

### 工牌背面

- Meaning: 邵宁 hides institutional authority when authority would hurt the person in front of him.
- First Appearance: S002.
- Active Scenes: S002.
- Payoff Target: later scene should force him to show the front again.
