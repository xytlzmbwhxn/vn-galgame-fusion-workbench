# S001 Rewrite V2 Readable Draft

值班室的白灯亮得发冷，交班表压在杯垫下面。

闸机屏又跳了一次红字。末班后的进站记录停在二十三点五十八分。

邵宁：证件。

岑遥：没带。

邵宁：那就写没带。

岑遥：别写。

邵宁：不写我就报警。

岑遥：那你报。

邵宁：电话在这儿。

岑遥：你先看纸的。

邵宁：看什么纸。

岑遥：登记。旧的。

邵宁：哪一年的。

岑遥：你们站里有几本。

邵宁：答年份。

岑遥：你先别开监控。

热水壶在角落里跳了一下，壶盖顶住蒸汽又落回去。

邵宁：你知道监控在哪儿。

岑遥：我知道它会少一段。

邵宁：谁教你的。

岑遥：吃过亏就会。

邵宁：你叫什么。

岑遥：写岑。

邵宁：名。

岑遥：先写姓。你不也把工牌翻过去了。
