# 《雨棚第七码》

## Genre / Tone

校园现实微悬疑 Galgame 短篇。雨夜、失物招领、轻喜剧表皮，底下压着一句不肯被本人认领的话。

## Player Role

程砚，南城大学失物招领雨棚的临时值班员。玩家通过扫码、验物、盖章、暂存、拆检等归还流程推动剧情。

## Core Promise

玩家每一次处理失物，都在决定一段关系该进入系统、留在私人手里，还是先被拖到下一分钟。

## Primary Mechanic

归还流程：

- 扫取件码
- 核对雨伞编号
- 检查伞柄暗格
- 给登记簿盖章
- 改成待核验

## Secondary Mechanic

证据与关系状态：

- `trust_lujin`
- `record_clean`
- `delay_clock`
- `rule_breach`
- `kept_recording`

## Route Structure

Branch-and-bottleneck。三种处理都会回到雨棚断电前的同一场对话，但保留不同状态，后续可扩为鹿今线、许栀线、制度线。

## Source Use Note

This project uses complete VN/open-source script corpora for structure and metrics only. No source dialogue or scene prose is copied.
