# Core Theme

## Thesis Question

一件失物被交还时，真正被认领的是物品，还是那个人不敢当面说出口的责任？

## Playable Thesis

玩家不能用一个纯粹正确的选项通关。每种处理都会牺牲一种东西：

- 干净记录会牺牲私人线索。
- 拆开暗格会牺牲流程时间。
- 暂缓核验会牺牲值班员的权限。

## Object Spine

- 七号长柄伞：表面是失物，实际藏着录音笔。
- 取件码：表面是归还凭证，实际让玩家决定谁有资格拿走声音。
- 红色印章：表面是手续，实际会把某句话盖成已处理。

## No Thesis Speech

正文不得让角色直接总结主题。主题必须通过扫码失败、盖章迟疑、语音外放、称呼变化、取件码失效来表现。
