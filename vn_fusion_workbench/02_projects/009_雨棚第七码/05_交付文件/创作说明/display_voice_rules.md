# Display And Voice Rules

## Readable Preview

- Dialogue rows render as `【角色】『文本』`。
- Thought rows render as `（文本）`。
- CSV stores raw text without quotes.

## Chengyan

- 名称：程砚。
- 句型：短到中句，偏流程词，干冷。
- 口吻钩子：把物件起行政外号，如七号伞、嗓门票、红章先生。
- 受压时：句子更短，称呼全名，手会先去找印章。

## Lujin

- 名称：鹿今。
- 句型：跳跃、撒娇、突然正经，喜欢问号和小声补丁。
- 口吻钩子：程值班、拜托嘛、欸、先别扫。
- 受压时：笑得更快，话更绕，手先按住要被检查的物件。

## Xuzhi

- 名称：许栀。
- 句型：轻、修正、自我剪辑，有广播稿式停顿。
- 口吻钩子：请删掉上一句、重来、第三遍。
- 受压时：不提高音量，反而更礼貌，更像在念通知。
