# Style Rules

## Dialogue

- 角色要比职业功能更活一点。
- 允许『』显示层、语气词、问号、感叹号、省略和半句，但必须服务声纹。
- 对话尽量成块推进，避免一句对话夹一句旁白再夹一句心理。
- 每句台词至少做一个动作：试探、岔开、求助、挑衅、降价、拖延、遮掩、示弱。

## Narration

- 旁白写屏幕上可见的东西：雨棚、扫码枪、登记簿、红章、伞柄、录音笔、备用灯。
- 少写抽象情绪，多写手、眼睛、物件位置、声音延迟。
- 不用结尾总结主题。

## Hard Stops

- 避免公式化转折句。
- 避免空泛的命运、救赎、意识到、复杂情绪。
- 避免只堆省略号作为沉默。
- 避免每行都以句号收束。
