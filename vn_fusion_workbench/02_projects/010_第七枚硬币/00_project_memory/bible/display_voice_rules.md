# Display Voice Rules

Readable preview:

- dialogue: `【角色】『台词』`
- thought: `（心理）`
- narration: 直接显示，不加角色名。
- choice: 用菜单项显示动作或一句可说出口的话。

Voice locks:

- 闻舟：机器状态、店规、温度、赔偿。句尾多句号，问号用于核事实。
- 夏停：嘴快、语气词、问号、省略号。痛处到来时句子断掉。
- 罗姨：电话外放、赔偿数字、生活化命令。骂完给实际办法。
- 活泼度底线：不管角色多冷、多累、多痛，台词都要保留一点可点击的趣味。闻舟用干冷机器比喻；夏停用昵称、撒娇和过快补笑；罗姨用生活化骂法和补救动作。

Punctuation:

- 省略号只给夏停和极少数停顿，不做空白三连。
- 感叹号主要给夏停和罗姨，闻舟慎用。
- 破折号用于闻舟截断旧事，不能滥用。
- 问号、语气词和未闭合句尾要属于角色本身，不为了统计而撒。
