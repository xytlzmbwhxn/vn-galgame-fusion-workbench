# 第七枚硬币

Genre / Tone: 现实微悬疑、午夜日常、轻玩法情感短篇。

Player Role: 闻舟，晚潮自助洗衣的夜班看店人，负责收尾、查口袋、处理机器故障和顾客投诉。

Core Promise: 玩家通过洗衣流程决定哪些痕迹被洗掉、哪些痕迹被留下，并承担留下痕迹带来的关系成本。

Central Pressure: 夏停想在天亮前洗掉一件开衫上的医院味，闻舟发现口袋里有会被水和热毁掉的热敏单。

Primary Mechanic: 洗衣流程选择：查口袋、投币、选温度、留单据、延长时间。

Secondary Mechanic: 痕迹变量：`trace_kept`、`receipt_saved`、`heat_damage`、`xia_trust`、`shop_risk`。

Reading Mode: ADV。

Route Structure: Branch-and-bottleneck。第一场选择影响开衫上的痕迹、夏停信任、后续她是否主动说出姓名。

Replay Value: 不同洗法保留不同证据和情绪入口；后续同一开衫会被重新解释。

Visual Motifs: 七号机、弯硬币、热敏纸、蓝塑料夹、医院腕带、滚筒里的水线。

Audio Motifs: 投币声、滚筒启动、烘干机低频、雨棚滴水、店主电话外放。

Ending Philosophy: 洗干净不等于放下。留下污点也不等于抓住真相。玩家要决定哪一种痕迹值得被明天继续看见。
