# Style Rules

- 文本显示按 Galgame/ADV 节奏写，每行是一个可点击状态。
- 对话用『』在可读稿中呈现；CSV 不手动加引号。
- 对话允许连续推进，心理描写集中出现在压力转折前后。
- 闻舟少用感叹号，夏停可用问号、省略号、语气词，罗姨可用电话式打断。
- 全角色默认 `liveliness +1`：任何人设都要比纸面设定稍微更有趣、更活一点。冷面角色要有干冷笑点，严肃角色要有生活小怪癖，悲伤角色也要有嘴硬、挑剔、撒娇、坏 timing 或小可爱。
- 每个主要角色每场至少出现一次可记住的“活泼钩子”：昵称、物件玩笑、奇怪比喻、错位礼貌、小抱怨、失败玩笑、语气词或可爱的身体反应。
- 环境描写必须动起来：投币、退币、滚筒、门圈、水线、电话外放。
- 不写空泛夜色，不写漂亮总结。
- 每 8-12 行至少出现一次可操作物或流程动作。
