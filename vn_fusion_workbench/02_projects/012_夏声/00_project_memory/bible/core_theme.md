# 核心主题

## 主题问题
为你伴奏的人，是在替你说你想说的话，还是在等你说这句话。

## 谱面作为角色隔离线
- 林知时的动作：弹、翻、记、不写名字。他在谱背面画一个重复符号（%），自己不签名。
- 唐叙的动作：折谱角、打叉、在有字的地方划双线。她写名字但不写日期。
- 许声远不在场，但他的笔迹留在三张谱上——全音符圈里写小字，不折页。

## 物件逻辑
- 钢琴：唯一能替人发声的东西，但下午五点要被抬走。时间不可逆转。
- 谱架：上面的谱子按新旧排开——合唱团两年换过三个伴奏。
- 第十页《城南送别》谱：唐叙毕业演出时用的那一页，背面有她写过的一句话。他拆下来放在谱架最下面，没告诉她。

## 情感锚
不存在"他爱她"的命名。存在的只有：
- 他在唐叙哑之后把她的独唱段转成钢琴独奏——在没人要求的情况下。
- 她在最后一页谱上打了一个叉，过了三天又回来把叉擦掉了大部分。
- 他记得她领唱的每一句，但说不出任何一个"她唱歌好听"这样的词。
- 她问他为什么不当领唱——他没有回答，而是弹了三个小节。
