# 主题母题

## 导入主题
- 替弹：林知时把唐叙毕业演出的独唱段改成了钢琴独奏——在她不在场、不需承担后果的情况下。这不是浪漫，是一种延迟的坦白——用琴键说一件用嘴不能说的事。
- 铅笔的传递：许声远→林知时→（未传）。唐叙认得这支铅笔，但她从来不问。三个人里只有一个人还在弹，一支铅笔连着三个人。
- 谱背面的字：唐叙写了"别换人"又擦掉——剩下的"别"+"半个'人'"成了这个场景里两人解读对方的核心证据。

## 活跃的物件主题
- 钢琴: 能替人说话，只能弹到五点。五点之后不能再弹了。是不是五点之后有些话就不能说了——这个压力贯穿全场。
- 铅笔: 许声远的牙印→林知时的手→唐叙的眼。物主变了，声音还在。
- 第十页谱: 背面写了又擦掉的字。被拆出来放在谱架最下面——他读了。她知道他可能读了。两人都不先说。
- 空乐谱夹: 唐叙带来的理由。没放任何东西进去。没放不进去。

## 可能的后续场景母题
- 如果林知时承认了铅笔是许声远的：唐叙是否会把它拿走——还是会留在琴凳上
- 如果唐叙承认了背面写的什么：林知时是否会用那个词——"别换人"——回答他把谱放到谱架最下面的原因
