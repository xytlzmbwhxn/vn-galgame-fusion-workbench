# S001 Readable Draft

琴凳盖没合上。

谱架上的谱子收了一半。最下一层单独放着一张手写谱——城南送别，第十页。

门上贴着搬家通知。下午五点。

（铅笔是谁的来着。）

（不是我的。但用了两年了。）

（她今天会来。）

清点清单。她没在清单上。

门没锁。他三点就来了，没锁门。

走廊里有脚步声。不是搬运工——太轻，而且不连。走两步停一下。

门口。唐叙站的位置刚好让门框挡住她半边。

【CHR_tangxu】『你在。』

【CHR_linzhishi】『清点道具。』

【CHR_tangxu】『五点钟搬家公司来。』

【CHR_linzhishi】『嗯。』

【CHR_tangxu】『我落了东西。』

【CHR_linzhishi】『什么东西。』

【CHR_tangxu】『一个乐谱夹。』

【CHR_linzhishi】『你手里那个空的。』

【CHR_tangxu】『不是这个。』

（她在找谱架最下面那张。）

（她不知道我十分钟前刚把它翻到面上。）

【CHR_tangxu】『你下午一直在弹。』

林知时把手指按在白键上——不弹。压着键不动

【CHR_linzhishi】『练指法。』

【CHR_tangxu】『不是指法。』

【CHR_tangxu】『是城南送别。』

【CHR_linzhishi】『听了多久。』

【CHR_tangxu】『从你升半音那里。』

（升半音是我改的。她毕业演出的时候原调太高——我没告诉她我降过。）

【CHR_linzhishi】『那首不好弹。』

【CHR_tangxu】『独唱段转成钢琴了。』

【CHR_linzhishi】『独唱段本来就该有人弹。』

【CHR_tangxu】『那天你不是伴奏。』

【CHR_linzhishi】『我在台下。第三排右边。』

唐叙的手指突然收紧了——在琴盖上。她不看他，看琴凳旁边他让出来的位置

【CHR_tangxu】『你站过那里。』

（她记得。她记得我在哪里。）

【CHR_linzhishi】『位置不挡门口。』

【CHR_tangxu】『那天你鼓掌了没有。』

林知时把铅笔从谱架边上拿起来——在指间转了一圈。不回答

【CHR_linzhishi】『那天我不是听众。』

（我弹完了。你不必知道。但你已经知道了。）

唐叙的目光落在那张谱上——不是谱面，是背面

【CHR_tangxu】『你翻到背面了。』

【CHR_linzhishi】『不是你留的。是纸褶。』

【CHR_tangxu】『我写的。』

她伸手碰到那张谱——不是拿，是指尖按在谱角的折痕上。折角还在

【CHR_tangxu】『后来擦掉了。没擦干净。』

【CHR_linzhishi】『剩了一个别。』

【CHR_tangxu】『还有一个字。』

【CHR_linzhishi】『人。』

【CHR_tangxu】『别……』

（她在等我说。但我说的不算。）

【CHR_tangxu】『你替谁弹的。』

【选择】

- 替我自己。替一首烂掉的歌。
- 沉默——弹了三个小节。不回答。
- 替许声远。这个教室应该有人替他把最后一张谱弹完。

【CHR_linzhishi】『替我自己。替一首烂掉的歌。』

【CHR_linzhishi】『你那天唱到一半的时候停了。我把后半句弹完了。不用替你——替我自己。一首没唱完的歌，我弹了。』

唐叙把谱从谱架上拿起来——不是收走，是放到琴凳上

【CHR_tangxu】『你弹得比我唱的好。』

【CHR_linzhishi】『琴不会哑。』

唐叙看着琴凳上的铅笔。认得。但没伸手

林知时没回答。手指放到键盘上——一个降号调，三个小节。独唱段的前三小节

琴声填了三小节教室。然后停。他没解释。

唐叙没有动。手指在琴盖边上停住——想摸键但他在弹

【CHR_tangxu】『这首……还有别的弹法吗。』

【CHR_linzhishi】『没有了。只剩这三个小节了。』

唐叙把谱从谱架上拿起来——折好——放进包里。不是透明夹

铅笔还在谱架边上。竖着。她没碰

【CHR_linzhishi】『替许声远。』

【CHR_linzhishi】『这个教室应该有人替他把最后一张谱弹完。他走的时候没弹。你领唱的时候替他弹了——毕业那天你替他唱了前半首。后半首是我替他弹的。』

唐叙把空乐谱夹放在琴盖上——是放在他能看见的地方。然后拿起谱架上的铅笔，端详牙印

【CHR_tangxu】『这支铅笔是他的。你一直在用。』

【CHR_linzhishi】『写了两年谱号。没写过一个名字。』

【CHR_tangxu】『他放在谱架上那天——说给下一个弹的人。下一个就是你。』

【CHR_linzhishi】『他说的不是我。他说的是伴奏。那时候我还没轮上。』

唐叙拿起第十页谱。看了背面的字——不再碰它。放回谱架——谱面朝上

【CHR_tangxu】『你弹完了。他该听到了。』

搬家公司的人来了。门上的通知被揭下来——他们照着电话确认的。

琴被抬上推车。琴凳收进车厢。谱架空了。

琴凳上只剩下那支铅笔。

唐叙拿起铅笔——不是揣进兜里。放在空乐谱夹里

【CHR_tangxu】『东西落下了。我找到了。』

琴走了。教室剩下一个空谱架。西窗打到地面——打在原来放第十页的位置。

谱架空了。铅笔竖着靠在谱架边上。

唐叙没拿铅笔。但在出门前停了——回头看谱架

【CHR_tangxu】『三个小节。你可以再多弹两个。』

琴走了。铅笔留在谱架边上——竖着，在琴原来在的位置前面。

第十页谱还在谱架上——谱面朝上。铅笔在空乐谱夹里。唐叙攥着夹子——合上了。

【CHR_tangxu】『谱留在这。铅笔我拿走。』

琴走了。第十页谱在空谱架上——面朝上。铅笔跟她走了。
