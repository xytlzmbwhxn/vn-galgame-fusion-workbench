# 风格参考包

本项目调用工作台已有方法：
- `galgame_textbox_production`：一行是一个受限视角文本框，服务演出而非散文展示。
- `dialogue_display_interiority`：对话和心理分层显示，口语可带标点能量与语气词。
- `line_energy_punctuation`：标点表达压力、遮掩、改口，不当装饰。
- `sentence_kernel_display_voice`：每个角色拥有句核和显示层口吻。
- `professional_ritual`：职业流程就是玩法动作，动作改变关系与证据。
- `theme_spine_playable_thesis`：主题通过重复玩家行为表达。
- `storylet_quality_state`：分支以状态留痕，回收时仍可见差异。
- `vertical_slice_production_pipeline`：首部短篇必须给出可验证切片。

本项目语感目标：
- 车站窗口的夜班疲惫、熟人式推诿、广播口吻、印章与热敏纸的实物感。
- 轻口吻下藏急事，不让角色一上来就讲完整苦衷。
- 悬疑线只露出可被窗口发现的部分：票号、班次、章、监控盲区、旧退票记录。
