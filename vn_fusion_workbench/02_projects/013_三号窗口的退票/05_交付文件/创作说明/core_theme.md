# 核心主题

主题问题：当一个人的痕迹只能通过窗口系统被承认，按流程做事还能不能留下人的体温？

可玩的主题行为：
- 玩家反复执行退票流程，把“拒绝”“待核”“异常”这些冷词变成可操作的选择。
- 每次盖章都改变关系状态：流程越完整，林桥越安全；备注越具体，孟雨越接近她要找的那个人。
- 关键物件是车票角。它同时是垃圾、凭证、风险源，也是孟雨不敢直接说出口的请求。

主旨落点：
- 这不是宏大救援，场景只允许一名窗口 clerk 做一件小事。
- 一枚章不能替任何人翻案，但能给下一次追问一个入口。
- 好作品的主旨要从操作里冒出来，不能靠人物站在原地讲道理。

短篇约束：
- 不写外部机关大战，不把真相一次说尽。
- 让玩家在“能留下什么”上做选择。
- 结尾保留后续空间，但本场景必须给出完整情绪回声。
