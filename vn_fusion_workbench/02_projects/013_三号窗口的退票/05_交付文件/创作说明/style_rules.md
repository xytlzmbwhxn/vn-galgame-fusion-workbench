# 文风与文本框规则

1. 对话可以连续，心理也可以连续。禁止机械地一人一句后插一句环境。
2. 每个文本框要带足可点击信息：事实、动作、态度、线索、风险、心理卡顿，至少占一项。
3. 对话口吻要能遮住角色名也认得出：
   - 林桥：流程词、短判断、干涩玩笑，把关心塞进操作。
   - 孟雨：轻快称呼、半句撒娇、临场改口，压力下把玩笑收掉。
   - 赵伯：车站值守口吻，粗、急、怕担责，话里带广播和巡检习惯。
4. CSV 原文不包 `『』`。可读稿由渲染器加 `【角色】『台词』` 与心理括号。
5. 旁白不抢戏，只写屏幕状态、物件、声音、手部动作和可演出变化。
6. 不用小说式长段外貌描写。人物状态靠动作、称呼、停顿、改口呈现。
7. 活用 `？`、`！`、`……`、`——`、无句号结尾，但每个停顿必须带隐藏词或策略变化。
8. 不要堆连续纯省略号，沉默交给 stage、sfx、表情或镜头。
9. 禁用均衡解释腔。尤其回避“不是 X，而是 Y”“不只是 X，也是 Y”“真正的 X 是 Y”。
10. 主题不在台词里自白。让退票流程、盖章风险、票角去向承担思想表达。
