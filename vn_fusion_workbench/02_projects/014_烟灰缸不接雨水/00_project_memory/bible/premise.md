# 《烟灰缸不接雨水》前提

## 作品名
《烟灰缸不接雨水》

## 类型/调性
现实微悬疑 / 凌晨自助银行 / 轻玩法情感短篇

## 玩家角色
陆川。自助银行夜班巡查维修工。凌晨三点被派来修一台吞了卡的七号ATM。三点半前必须修好，否则安保系统自动报警。

## 第一场地点
24小时自助银行的玻璃隔间。外面暴雨。灯管白得发冷，地砖上印着前一拨顾客的湿脚印。

## 核心承诺
玩家通过维修流程、监控规则、物品操作来决定一个陌生人的等待是否值得被保护。

## 中央压力
ATM隔间有摄像头每15分钟抓拍一次。女孩站在盲区里——她知道这个规则。陆川必须在"按流程上报"和"帮她多站一会儿"之间选。

## 主题
人不一定每次都敢把关心说成关心。有时候说成"设备检修"，说成"工具箱你可以坐"，说成"规定上没说不能多留一盏灯"。
