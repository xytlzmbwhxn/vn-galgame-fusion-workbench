# 《烟灰缸不接雨水》文风规则

## 写作契约

1. 对话必须连续推进，不机械交替对白/心理/场景。
2. 每一行为一个点击单元，承载清楚的动作、反应或信息变化。
3. 心理描写必须是角色的判断、顾虑、记忆或自我欺骗，不把纯场景描写伪装成心理。
4. 人物永远比默认设定更有趣一点：陆川的冷里藏着笨拙关照，施遥的轻快里藏着不敢问出口的害怕。
5. `『』`框对白，`（）`框心理。旁白和环境不加括号。
6. 避免AI腔、概念化说教、空泛转折句。
7. 演出标注服务制作，正文服务玩家。环境描写短而有用，不抢对白节奏。
8. 活用标点和语气词：`？` `……` `嘛` `欸` `吧`，句号不要每句必有。
9. 句子长短错落。独词句只在画面或声音支撑时使用。
