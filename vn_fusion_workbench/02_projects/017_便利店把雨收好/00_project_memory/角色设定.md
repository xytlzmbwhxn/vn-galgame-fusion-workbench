# 角色设定

## 江迟

- 玩家视角，便利店夜班店员，刚从学校退掉社团职务，怕麻烦，也怕别人认真谢他。
- 公开想要：把班平安交完，不被店长骂。
- 私下想要：有人能承认他“留下来”这件事有意义。
- 说话指纹：短句偏多，遇到情绪会先报流程；心理行会把情绪翻译成库存、班表、小票、规章。
- 压力反应：先装作公事公办，后来会把规则往人身边挪一点。
- 禁区：别人说他“好人”“可靠”。

## 林檐

- 常来便利店躲雨的高三复读生，伞柄缠着透明胶，嘴硬，反应快，把尴尬打成玩笑。
- 公开想要：找回一把被她弄丢的蓝伞。
- 私下想要：确认自己没有把母亲留下的东西弄丢到再也找不回来。
- 说话指纹：疑问多，句尾喜欢轻轻挑一下；被戳中会用“喂”“好嘛”“那你很会哦”转移。
- 压力反应：先挑刺，再撒小谎，最后用很小的声音补真话。
- 可爱钩子：把便利店的设备起外号，收银机叫“铁算盘”，打印机叫“吐纸兽”。

## 何姨

- 白班店长，未出场，只通过电话和规章存在。
- 功能：把“流程”变成压力，同时让江迟有可以躲的挡板。
