# 对白所有权重写计划

## 重写目标

旧稿主要问题：大量长对白块保持等权互怼，虽然口吻比早期活泼，但缺少临时说话权，导致场面像聊天记录。

新版目标：

- 保留便利店、蓝伞、失物登记、豆浆、午休回调。
- 减少默认 A/B/A/B。
- 让关键段落由一个角色连续持有压力。
- 长连续对白保留，但必须有功能。
- 心理和旁白放在对白块之后作为铰链。

## Dialogue Ownership Plan

### Block 1：进店与蓝伞说明

- Block goal：让林檐以连续跑偏的玩笑遮住真正焦急。
- Pressure owner：林檐。
- Same-speaker run owner：林檐，4 textbox。
- Why multiple clicks：她从“别欢迎”一路把便利店设备、雨、蓝伞、牙印讲出来，每格都降低一点自尊防线。
- Interruption point：江迟用登记流程打断。
- Narration/thought hinge：江迟看见她先看失物箱，判断她绕开了购物目的。
- Anti-ping-pong move：前半由林檐连说，后半由江迟流程连说。

### Block 2：登记与选择

- Block goal：把“找伞”变成玩家第一次站队。
- Pressure owner：江迟。
- Same-speaker run owner：江迟，3 textbox。
- Why multiple clicks：他连续报流程、特征、后果，像挡板，也像靠近。
- Interruption point：林檐嫌“牙印”太丢脸。
- Long dialogue function：手续感制造情绪成本。

### Block 3：监控与小票

- Block goal：用小票和监控把失物变成可操作玩法。
- Pressure owner：江迟先持有流程，林檐随后持有情绪。
- Same-speaker run owner：江迟 3 textbox，林檐 4 textbox。
- Why multiple clicks：一边是权限/隐私/证据链，一边是她终于承认蓝伞的家庭意义。
- Narration/thought hinge：小票出现后，江迟意识到她没有求他，只是把“今天”放在柜台上。

### Block 4：借伞与离店

- Block goal：让江迟第一次主动把规则往她那边挪。
- Pressure owner：江迟。
- Same-speaker run owner：江迟，2-3 textbox。
- Interruption point：林檐拒绝欠人情。
- Long dialogue function：借伞把善意说明压成关系债签订。

### Block 5：第二天午休

- Block goal：支付前一晚分支债。
- Pressure owner：林檐。
- Same-speaker run owner：林檐，连续说出父亲、新伞、旧伞和寄存。
- Narration/thought hinge：江迟不解释主题，只把失物本改成寄存本。
