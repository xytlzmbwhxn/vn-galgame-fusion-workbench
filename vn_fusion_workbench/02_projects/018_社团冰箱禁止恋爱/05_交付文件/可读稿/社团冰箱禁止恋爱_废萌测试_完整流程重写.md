# 社团冰箱禁止恋爱_对白块重写

## 生成前计划

- 默认视角：顾淮。
- 配角心理：仅在结尾使用 @pov 夏梨，制造玩家知道、顾淮不知道的小糖点。

- Dialogue ownership plan：开场由夏梨占压，她用冰箱拟人和布丁申诉打开场面；顾淮用财务流程连续接住。
- Same-speaker run owner：夏梨 3-5 格连打负责废萌能量；顾淮 2-3 格连续流程负责反差。
- Interruption point：顾淮在夏梨准备“额头盖章”时打断。
- Thought hinge：只在发现空位、确认贴纸、收到“暂时禁止”时进入顾淮脑内。
- Anti-ping-pong move：避免长期 A/B/A/B，重要段落用同角色连续推进。
