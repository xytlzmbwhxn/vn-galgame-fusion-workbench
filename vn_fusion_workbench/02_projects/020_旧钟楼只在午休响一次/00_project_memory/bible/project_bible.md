# 旧钟楼只在午休响一次

## 类型

校园废萌 / 轻异常 / 午休短篇。

## 可玩承诺

玩家扮演总被临时委托的钥匙保管员，在午休里处理一口“只多响一下”的旧钟。每次选择不是解谜胜负，而是决定你把麻烦接成谁的借口、谁的秘密、谁的甜点债。

## 软开头原则

开场不使用“我叫某某”的硬身份牌。身份从旧钟楼、失物箱、钥匙串、他人称呼和临时任务里浮出来。

## 语气目标

- 日常轻松，角色比设定更活一点。
- 反问、放松、语气词、停顿和半句要从角色当前压力里长出来。
- 心理行只写镜头拍不到的判断、犹豫和选择成本；物件状态放旁白或 stage。

## 核心物件

- 旧钟楼钥匙：谁拿着它，谁就会被大家默认为“能负责的人”。
- 12:17 的多余钟声：像异常，也像某个人给午休加的一枚书签。
- 焦糖布丁券：温眠的甜点债，也是钟声事件的社交筹码。

## 本场交付目标

S001 是完整可读 vertical slice：软开头、角色引出、三人对白块、一个选择、三条短分支、合流和后续钩子。
