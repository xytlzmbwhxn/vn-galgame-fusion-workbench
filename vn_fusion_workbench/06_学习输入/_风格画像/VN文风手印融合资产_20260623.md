# VN 文风手印融合资产

这份文件是公开仓库用的可调用风格资产。它保留可执行写作手感，不记录具体参考作品名、私有来源路径或原文样本。

## 语料基准

以下指标只作为点击节奏校准，不是模板：

- 对话比例通常可以高于叙述比例。
- 成熟 VN 的单个 textbox 往往较短，但短不等于低信息量。
- 长对白块、同说话人连续多格、短促顶嘴都可以成立，关键是每次点击必须推进压力。
- 可见选项、隐藏条件、跳转和回调都应服务“玩家做了什么被记住”。

## 文风原则

- 成熟 VN 的文风不是华丽句子，而是点击后的关系变化。
- 日常不是水，是低烈度摩擦、误会、欠债、回调的储能层。
- 可爱不是语气词堆叠，而是角色在压力下露出的小偏差。
- 心理描写不是解释心情，而是让玩家看见角色不敢说的选择成本。
- 旁白要服务屏幕、节奏和误读，不要替玩家总结主题。
- 标点是角色的呼吸，不是统一美化包。

## 句法动作库

### 轻口角转关系债

- 形态：玩笑或挑衅先行，随后出现一个小动作或小让步，让玩家意识到关系比台词软。
- 使用：开场、日常、关系升温、喜剧后回落。

### 错误理解转私密判断

- 形态：角色说一句表层话，主角先误读，再通过一个具体细节修正判断。
- 使用：心理口袋、选择前、发现角色面具裂缝。

### 连续追问压出真意

- 形态：两到四个短对白连续推进，不插解释；到最后一行让对方无法继续装傻。
- 使用：冲突、调侃、告白前、路线锁前。

### 日常手续承载情绪

- 形态：通过付款、登记、排队、归还、打卡、维修、取物、售后、复核等手续暴露关心或逃避。
- 使用：本土日常、校园社团、职业场景、家庭压力、轻喜剧暧昧。

### 沉默后不解释

- 形态：沉默用演出或动作承接，下一句说实际问题，不把沉默翻译成主题。
- 使用：伤感、尴尬、迟疑、关系变质。

### 句尾泄露

- 形态：角色想维持姿态，句尾却软掉、急掉、断掉或改口。
- 使用：可爱感、嘴硬、害羞、强撑。

## 对白块动力学

- 不要默认写成“你一句我一句”。
- 对话块要先确定“说话权”：谁暂时拥有压力，谁被迫接招，谁负责打断。
- 一个角色可以连续占两到五个 textbox，但每一格要改变策略、信息、情绪泄露、关系成本或对象操作。
- A/B/A/B 适合短促顶嘴、喜剧冲撞、快速打断；长段必须加入同说话人 run、动作打断、误读、转移话题、称呼变化或心理口袋。
- 心理和旁白优先放在对白块之后作为推理/屏幕状态的铰链，不要每句后面都插。

生成前必须写：

```text
Dialogue ownership plan:
Block goal:
Pressure owner:
Same-speaker run owner:
Why this run needs multiple clicks:
Interruption point:
Narration/thought hinge after block:
Anti-ping-pong move:
Long dialogue function, if any:
```

## 心理描写风格

允许：

- 短促私问
- 自我否认后立刻被事实打断
- 对对方话语的危险解读
- 想做与应该做之间的卡顿
- 对前文分支债务的记忆
- 现实小担心里夹着更软的情绪
- 先误读，再被一个具体细节修正
- 句子停在不敢说出的称呼或名词前

禁止：

- 摄像机能拍到的动作
- 漂亮但无决策压力的比喻
- 解释主题
- 重复上一句对白的意思
- 连续三行以上内心独白
- 用固定对称句式拆句规避
- 把心理行写成审稿意见、人物分析或主题注释

内心口袋节奏：

- 通常 1 行成立，2 行可以形成推理和代价，3 行必须通向选择、路线锁或不可撤回的回复。
- 对白块之后再给心理，比每句台词后面塞心理更有代入。
- 心理行要让下一次点击更难、更近或更危险；如果只让句子变漂亮，删掉。
- 机械心理的修复方向：把作者判断换成主角的误读、旧债、称呼变化、可选行动、害怕被问到的名字。

## 废萌 / 轻喜剧特别规则

- 角色要比设定高半档可爱，但不能像卖萌台词生成器。
- 可爱来自小失败、临时嘴硬、起外号、把害羞讲成业务、把照顾讲成费用。
- 心理描写少而准：不解释喜欢，只暴露“想多看一秒但找不到理由”。
- 旁白不抢戏，主要负责屏幕、手部动作、距离、物件状态。
- 句子可以活泼，标点必须属于角色；不要全员平均使用“？！”。 

## 角色声音原型

- `high-energy comic pressure`：高疑问、高感叹、短促反应制造场面推力；底层要藏自尊、怕输、怕被丢下或怕被看穿。
- `quiet relation pressure`：短句、沉默、日常确认制造关系重量；必须给她具体行动目标。
- `hesitant soft pressure`：省略、低声确认、退半步的句尾制造温柔和不安；不要让角色只会软。
- `adult loud pressure`：大声、玩笑、行动催促推进场面；真话藏在吵闹后面。
- `procedural affection`：用登记、售后、复核、编号、费用等流程词隐藏关心。

## 生成协议

1. 先定角色当下的难处：想要什么，不敢说什么，手里拿着什么。
2. 再定场面手感：轻喜剧、撒娇、尴尬、嘴硬、迟疑、被戳中。
3. 再写对白块：允许同一角色连续占两到五格，但每格必须换策略、信息、情绪泄漏或关系成本。
4. 再放心理口袋：只写镜头拍不到的东西，且让下一次点击更难。
5. 最后才扫禁令：删机械句式，保留有角色位置、有口语动机、有场面节奏的自然转折。
