# Public CN Gal Tutorial Style Profile

公开版教程风格画像，只保留可执行写作规则，不记录具体教程来源。

## 生成规则

- 开头用问题、环境、物件、规则故障或心理漏点引出，不用硬自我介绍。
- 对话、心理、客观事实分块，不要每句对白后面塞一句心理。
- 心理通常放在对白块之后，作为推理、屏幕状态、选择压力的铰链。
- 配角心理必须有明确 POV 切换；默认不要偷变全知。
- 长对白必须安排对白所有权：谁占压、谁躲、谁打断、哪里转场。
- 每个选择都要留下状态：物件、关系、信息、时间、路线或称呼。

## 废萌/日常规则

- 日常段落也要有小目标：收东西、借东西、登记、赔偿、复核、等人、藏纸条。
- 轻喜剧台词要让角色位置更清楚，不只是变可爱。
- 语气词、反问、放松拍、半句撤回都可以用，但必须属于角色。
- 心理描写要像当下冒出来，不像作者总结。

## QA

- 去掉机械“不是……而是……”结构。
- 去掉主题总结句。
- 去掉无选择压力的漂亮比喻。
- 合并低负载短点击。
- 补足物件、动作、称呼、关系债和后续回调。
